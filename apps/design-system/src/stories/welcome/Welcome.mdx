`@yeong/ui` 스토리북에 오신 것을 환영합니다! 🎉  
이 스토리북은 재사용 가능한 UI 컴포넌트를 확인, 문서화 및 테스트하기 위하여 작성되었습니다.

## `@yeong/ui` 란?

`@yeong/ui`는 [Jii-Yeong/yeong](https://github.com/Jii-Yeong/yeong) Repository 안 프로젝트에서 쓰이는 모노레포 내부 패키지 입니다.

## 시작하는 방법

- **사이드바**에서 사용 가능한 컴포넌트를 살펴보세요.
- **Docs** 탭을 이용하여 컴포넌트 사용 방법 및 API를 확인합니다.
- **Controls** 탭을 통해 props를 조종하고 컴포넌트의 동작을 테스트합니다.
