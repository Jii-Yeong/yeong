## Usage

CommonDivider 컴포넌트는 기본적으로 border로 되어있기 때문에 색상값이나 스타일, 굵기를 지정할때는 tailwind css border 속성을 부여하면 됩니다.

### 예시

```tsx
<CommonDivider className="border-red border-[2px] border-dotted">
```
